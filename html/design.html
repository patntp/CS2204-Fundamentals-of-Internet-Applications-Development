Page Design

CW1

Common information in all pages (except Design Page)
  • a) img tag is used to get the logo image, h1 tag is used to show the cinema's name, and header tag is used to group those content as the header.
  • b) a tags are used to provide hyperlinks to other pages, and nav tag is used to group a tags together as main menu for navigation.
  • c) p tag is used to give copyright information, a tags are used to provide hyperlinks to other pages, and footer tag is used to group those tags together as the footer.
Home Page
  • d) video tag is used to get video file, h3 tag is used to show movie's name and information, and section tag is used to group those tags together.
  • e) h3 tag is used to show the topic, dl tag is used to create a list containing 2 dt tags for 2 sub-blocks, li tags are used to list each movie's information which has img tags to get image files, and section tag is used to group those tags together.
Now Showing Page
  • f) label tag is used to tell the purpose of a particular input, and select tag is used to provide the cinema drop down list.
  • g) ul tag, and li tag inside are used to create a list of movies, input tag is used to show movie's name in the grid(read only) and create Buy Ticket button(type submit), img tag is used to get image file, select tag is used to provide the show time and house drop down list, form tag with get method is used to send data to Ticket Page attached to url, and section tag is used to group f),g) together.
  • h) dl tag is used to create a list, dt tag is used to give the title; Cinema Locations, li tags are used to list the map(img tag) and address(p tag), and section tag is used to group those tags together.
Ticket Page
  • j) h3 tags are used to show information of selected movie.
  • k) table tag is used to create the seat plan with tr tag to create each row, td tag to create the screen area and each seat in a row, and tbody to group tr and td tags together.
  • l) p tag is used to tell selected seat, input tag is used to create Confirm button(type submit), fieldset tag is used to group those information into a form, legend tag is used to give description of the fieldset, form tag is used to send data to Print Page (for future assignments), and section tag is used to group j),k),l) together.
Print Page
  • m) label tag is used to tell the purpose of a particular input, input tag is used to show information in the grid(read only), section tag is used to group each ticket's information together and separate two tickets.

Requirement No.

  • 18) If css of design page is counted, then 6 style sheets and 2 groups of styles(embedded style and external style) are used. Following the application functions approach, wholesite.css is the highest level; it controls common layout, color scheme, and font types for all pages. Other 5 style sheets are in charge of the layout and requirements for a particular page. One embedded style is used in print page to meet the print preview requirement because this css script handles very specific requirement and its length is short.
  • 19) 2 CSS3 features are found in wholesite.css; transition(line 31-37) and round border corner(line 50).
